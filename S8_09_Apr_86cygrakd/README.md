## Installation
1. Clone the repository:  
   ```bash
   git clone https://github.com/your-repo/todo-app.git
   ```
2. Navigate to the project directory:  
   ```bash
   cd todo-app
   ```
3. Install dependencies:  
   ```bash
   npm install
   ```
4. Start the development server:  
   ```bash
   npm start
   ```
